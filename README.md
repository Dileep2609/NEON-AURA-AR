# 🌌 NEON AURA AR

An interactive Augmented Reality hand-tracking experience built using **MediaPipe Hands**, **HTML5 Canvas**, **JavaScript**, and modern visual effects.

Users can interact with dynamic neon particles, gesture-based shockwaves, and customizable themes using real-time hand tracking through their webcam.

---

## 🚀 Features

### ✋ Real-Time Hand Tracking

- Detects up to 2 hands simultaneously
- Tracks all hand landmarks
- Powered by MediaPipe Hands

### 🌈 Dynamic Neon Themes

Choose between:

- Rainbow
- Cyberpunk
- Ocean

Themes instantly update all visual effects.

### ✨ Particle Effects

- Neon particle trails
- Finger-tip glow effects
- Dynamic particle physics

### 💥 Gesture Recognition

- Pinch Detection
- Open Hand Detection
- Hand Spread Percentage

### 🌊 Shockwave Effects

Create animated energy ripples when pinch gestures are detected.

### 📊 Live HUD

Displays:

- Number of Hands Detected
- FPS Counter
- Current Gesture
- Hand Spread Percentage

### 🎥 Camera Integration

- Uses webcam feed
- Mirror mode enabled for natural interaction

## ▶️ Usage

1. Open the application.
2. Click **Enter Experience**.
3. Allow camera permissions.
4. Move your hand in front of the webcam.
5. Try pinch gestures to trigger shockwaves.
6. Switch themes using the theme buttons.

---

## 🎮 Supported Gestures

### Pinch Gesture

Bring thumb and index finger together.

Result:

- Gesture changes to PINCH
- Shockwave effect is created

### Open Hand

Keep fingers apart.

Result:

- Gesture changes to OPEN
